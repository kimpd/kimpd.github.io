개발내용 :

같은 디자인의 템플릿 페이지가 반복되는 인쇄물(상품 카탈로그, 앨범 등)의 자동편집 시스템 개발.
인디자인의 스크립트 기능을 활용하여 대량의 이미지, 텍스트 리소스를 자동으로 배치하여 가편집 상태의 편집물을 완성한다.

시스템구성 :

1. Beyond Ruler - 디자인 템플릿의 이미지 박스, 텍스트 박스의 편집 설정 프로그램.
   (Windows 7/10, C#, WPF)

2. Beyond Script - 어도비 스크립트로 인디자인의 전체 기능을 단계적으로 자동화하여 편집.
   (MacOSX, InDesign CC 2015, Adobe Script)

3. Beyond Resource - 웹서버를 통한 리소스(이미지, 텍스트)의 관리.
   (자동편집시 설정된 폴더구조에 맞게, 로컬로 다운로드하여 리소스를 사용하는 방식)
