개발내용 :

페이지 템플릿과 기능 컴포넌트를 기반으로 하는, 쇼핑몰사이트 제작 자동화 솔루션 개발.
기존 개발된 사이트빌더에 쇼핑몰에서 필요한 페이지와 컴포넌트, 결제기능을 추가 개발함.

시스템구성 :

1. 쇼핑몰 빌더 솔루션.
   (Linux, WebObjects 5, Java, EOF, OracleDB)

2. 웹어플리케이션, 서버관리는 웹오브젝트 관리자 기능에서 제공.
