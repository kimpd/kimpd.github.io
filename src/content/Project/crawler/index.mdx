개발내용 :

웹상의 정보들을 탐색, 분류, 수집하는 작업을 자동화하여,
기존의 솔루션들과 연계하여 다양한 형태로 적용될 수 있는,
웹 스크래핑(Web Scraping), 웹 크롤링(Web Crawling) 도구 개발.
현재 Windows, MacOSX, Linux 환경에서 수집목적에 맟춰 적용 가능.

시스템구성 :

1. Windows - 윈도우 전용 데이터 수집 프로그램으로 개발되어, 화면상의 마우스를 제어하여 실행된다.

   이미지 분석 및 수집에 적합.
   (Windows 7/10, C#, .Net, Selenium)

2. MacOSX - 병렬처리와 안전성을 위하여 개발된 Golang 콘솔모드 크롤러.

   대량의 데이터 수집 및 많은 시간이 걸리는 작업을 백그라운드로 처리.
   (MacOSX, Golang, Colly, chromedp)

3. Linux - 웹브라우저에서 동작하는 웹서비스 형태의 스크래퍼.

   간단한 구조를 가진 사이트의 내용과 자료 등을 수집하는데 적합.
   (Linux, Node.js, JavaScript, Puppeteer)
